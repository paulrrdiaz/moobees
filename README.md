## Commands

### INSTALL: It'll install all dependencies

```bash
npm i
```

### START: it'll start http://localhost:3000

```bash
npm run start
```

### BUILD: it'll prepare the bundle

```bash
npm run build
```

### CONTAINER: it'll start http://localhost:5000

```bash
docker build -t moobees .
docker run -it -p 5000:80 moobees
```
